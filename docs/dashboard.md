# Dashboard

The first thing you see after signing in to the [[platform-overview|teacher platform]].

- **Three stat tiles** — Classrooms, Students, Resources — each a shortcut to that page.
- **Your classrooms** — the first few classes as cards (name, level, description, roster size, term); click through to [[classrooms-and-students|Classrooms]] for the full list.
- **A Knowledge card** — a short pointer into the [[knowledge-graph|Knowledge]] view, the graph visualisation of your whole workspace.

There's nothing to configure here — it's a landing view built entirely from data you enter elsewhere (classrooms, students, resources). If a number looks wrong, the fix is always on that entity's own page, not the dashboard.
